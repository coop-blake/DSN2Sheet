Windows Command Line Utilities

A set of command line utilities for moving data from an ODBC Data source to either a remote Google sheet or local text file.

Unsigned binaries built by github actions can be found in release.

You can also build your own from the repository after installing rust with the following command.

cargo build --release

This will create the two programs outlined below

DSN2Sheet - Push a DSN Query to a Google Sheet
DSN2Text - Save DSN Query as text file
