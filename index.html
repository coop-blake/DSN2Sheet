Windows Command Line Utilities

A set of command line tools for querying an ODBC data source and exporting the results to a Google Sheet or saving them in a local text file.

Unsigned binaries built by github actions can be found in releases.

You can also build your own from the repository

DSN2Sheet - Push a DSN Query to a Google Sheet
DSN2Text - Save DSN Query as text file
